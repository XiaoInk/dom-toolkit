# DOM Toolkit - 项目总览

## 📋 项目简介

DOM Toolkit 是一个轻量级的JavaScript库，专门用于简化浏览器DOM操作。通过基于坐标的精确控制和模块化设计，为网页自动化测试、数据采集和用户交互模拟提供强大而简单的解决方案。

## 🎯 核心价值

### 🎯 精确控制
- **像素级精度**：基于坐标的精确定位和操作
- **实时反馈**：坐标显示和元素检测功能
- **多模式交互**：支持点击、输入、拖拽等多种操作

### 🧩 模块化设计
- **按需加载**：4个独立模块，支持单独使用
- **统一接口**：提供简洁的API外观
- **无依赖设计**：纯原生实现，无外部依赖

### ⚡ 极致性能
- **轻量级**：完整包仅2.09 kB
- **高性能**：优化的DOM操作和事件处理
- **内存友好**：完善的资源清理机制

### 🛠️ 易于使用
- **零配置**：开箱即用
- **多场景支持**：控制台、CDN、本地部署
- **完整文档**：详细的使用指南和示例

## 📦 核心模块

### 📍 坐标显示模块 (domToolkitCoordinates)
实时显示鼠标坐标，帮助精确定位页面元素。

### 🔍 元素检测模块 (domToolkitDetector)
通过坐标获取元素信息，支持高亮显示和实时检测。

### 🖱️ 交互操作模块 (domToolkitInteractor)
模拟用户交互操作，包括点击、输入、拖拽等。

### 📜 滚动控制模块 (domToolkitScroller)
精确控制页面滚动，支持多种滚动模式和批量操作。
