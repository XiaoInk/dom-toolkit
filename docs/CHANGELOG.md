# CHANGELOG

## [1.0.0] - 2025-01-14

### 🎉 初始发布

#### ✨ 新功能
- **坐标显示模块** (`domToolkitCoordinates`)
  - 实时显示鼠标坐标
  - 可关闭的坐标显示器
  - 页面滚动位置获取

- **元素检测模块** (`domToolkitDetector`)
  - 通过坐标获取元素详细信息
  - 元素高亮显示功能
  - 实时元素检测模式（Ctrl+鼠标移动）

- **交互操作模块** (`domToolkitInteractor`)
  - 单击/双击/右键点击
  - 文本输入和表单填写
  - 拖拽操作
  - 批量操作支持

- **滚动控制模块** (`domToolkitScroller`)
  - 方向滚动（上/下/左/右）
  - 位置滚动（到顶部/底部/指定坐标）
  - 相对滚动
  - 滚动到元素
  - 复杂批量操作序列

#### 🏗️ 架构设计
- **模块化架构**：4个独立模块，可按需加载
- **统一API接口**：`domToolkit` 全局对象
- **Promise基础**：所有异步操作返回Promise
- **IIFE封装**：兼容性良好的模块格式

#### 📦 构建系统
- **多种构建目标**：
  - 完整工具包 (`dist/index.js`)
  - 控制台注入版本 (`dist/dom-toolkit.inject.min.js`)
  - 独立模块 (`dist/modules/*.min.js`)
